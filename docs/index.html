@etomon/ioredisdown

Etomon Search Client

The project provides a portable library for searching for Courses in our backend.

Examples of using this library include: displaying search results based on a keyword, listing courses as a dropdown, or for rendering courses as a list in the admin panel.

Structure

EtomonSearchClient

API/EtomonSearchClient provides methods for performing searches on our backend via WebSocket. It is a drop-in replacement for the old public/search.js library and is 100% compatible with the old functionality.

This API client can be used both on the server-side and client-side.

GoogleLocationAutocomplete

Building

Etomon Search is written in TypeScript. The UI Component is built on React, and so is written in TSX (like JSX but TypeScript).

The API client depends on socket.io.

To build run npm build.
